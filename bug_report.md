# 仕様
- 一人のユーザーは，ひとつ商品につきひとつのみ購入可能
- 購入した商品は，再度購入は不可能
- ログイン前にカートへ追加した商品は，ログイン後にも適切に保持されている
* 基本的に現状の動きを大きく変えないようにすること

# 注意
今回のサイトでは、カートに入ったものを購入する際の処理（入金や配送先の入力）は省くものとする。
したがって、カートから購入ボタンを押したら購入は完了するものとする。
